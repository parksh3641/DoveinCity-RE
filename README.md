### 02. Scripts
* **Default** : 공통적으로 사용되는 스크립트
* **Manager** : 각각의 씬(Scenes)을 관리하는 스크립트
* **Game** : 인게임화면에서 사용되는 스크립트
* **UI** : 메인화면에서 사용되는 스크립트
* **Tutorial** : 튜토리얼에서 사용되는 스크립트
* **Network** : 멀티플레이에서 사용되는 스크립트
* **Google** : 구글 로그인, 리더보드, 업적, 구글 애드몹(광고) 등 구글 플러그인을 관리하는 스크립트
